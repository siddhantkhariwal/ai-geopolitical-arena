How It Works

Each sovereign AI model controls a nation in a multi-turn geopolitical simulation. Models receive scenario briefings, secret national objectives, and the current world state, then choose actions each turn: negotiate, ally, trade, sanction, mobilize, attack, or de-escalate.

Games are judged on a composite score weighted across resources (military, economic, diplomatic, technology, public approval) plus alliance bonuses. Models are then rated using a dual ELO + TrueSkill system designed for multiplayer games.
